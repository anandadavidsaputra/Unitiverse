# 🌍 Unitiverse

Unitiverse adalah website konversi satuan yang dirancang untuk memudahkan pengguna mengubah berbagai jenis satuan ukur secara cepat dan praktis dalam satu platform.

---

## 🌐 Live Demo

👉 [https://anandadavidsaputra.github.io/Unitiverse/](https://anandadavidsaputra.github.io/Unitiverse/)

---

## 🧩 Overview

Unitiverse dibangun sebagai proyek latihan pengembangan web untuk:

* mengonversi berbagai jenis satuan ukur
* menyediakan kategori konversi yang beragam
* menampilkan hasil konversi secara instan
* melatih implementasi logika perhitungan menggunakan JavaScript

Kategori konversi yang tersedia meliputi:

* 📏 Panjang (mm – km)
* ⚖️ Berat (mg – ton)
* ⚗️ Volume (ml – m³)
* ⏱️ Waktu (detik – tahun)
* ⚡ Energi (joule – kalori)
* 🌡️ Suhu (Celsius, Fahrenheit, Kelvin)
* 💾 Data (bit – gigabyte)
* 🔭 Astronomi (tahun cahaya dan satuan jarak astronomi lainnya)
* 🚀 Kecepatan (m/s – km/jam)

Website ini menggunakan struktur menu terpisah untuk setiap kategori konversi agar lebih terorganisir dan mudah digunakan.
